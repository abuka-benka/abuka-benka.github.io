前置知识：class(类)

py 规定每个类都要继承自另一个类，而最终所有类都必须继承自 object 。相对的，就有了基类和子类。这是一个相对的概念，一个基类也可以是另一个类的子类。

任何类的祖宗类都是 object

抽象类，是不可被实例化，只能被继承的基类，如 dataset。

抽象类提供了一个模板，规定了子类的实现方法，从而为他的子类提供了一个接口,如只要是继承自Dataset的子类，都可以被 pytorch 的函数识别为数据集。
