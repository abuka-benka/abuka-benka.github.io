hexo 指令

功能分类 指令 说明
初始化 hexo init blog 创建新博客
npm install 安装依赖
文章/页面 hexo new "标题" 新建文章
hexo new page "about" 新建页面
本地调试 hexo s 启动本地服务器，默认端口 4000
hexo s -w 启动并监听改动，实时刷新
hexo s -p 5000 指定端口启动服务器
生成/部署 hexo g 生成静态文件
hexo d 部署到 GitHub Pages
hexo g -d 生成 + 部署（简单更新）
hexo clean 清理缓存（缓存问题、页面不更新时用）
组合操作 hexo s -w 写作 + 本地预览
hexo clean && hexo g -d 清理 + 生成 + 上传（彻底更新）
其他 hexo help 查看命令帮助
hexo -v 查看 Hexo 和插件版本

渲染冲突问题

butterfly 的数学公式渲染插件很迷啊。。。怎么安都不行，结果试了七八种插件，越改越乱，直接在 blog 文件夹里养蛊了。最后 chatgpt 救我狗命。

调整卡片透明度

butterfly 的配置里是没有这个选项的，但我认为黑夜背景配上白色卡片很丑，因此在 chatgpt 的指导下更改了 .pug 文件（负责生成html）。反复调试后大胜利。

刚开始尝试直接修改透明度，发现 butterfly 的文字和卡片是在一个图层里。因此需要先将文字设置为悬浮，然后再设置卡片透明度。
